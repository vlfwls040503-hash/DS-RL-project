# 합류·분류부 통합 — 전체입력 모델 (신월여의 + 복층터널)

> 합류/분류 구간 주행로그를 통합해 **전체입력(도로+자차상태+조건) 결정론 GRU**를 학습. 피실험자 단위 held-out 평가.

## 데이터
- 통합: 신월여의(본실험+추가) + 복층터널 지하도로
- 피실험자 **155명**(인물 파싱), 총 **328만 행**, 18 특징
- 분할: train 111명 / val 22 / test 22 (행: 234만 / 51만 / 43만)

## 결과 (held-out 피실험자, 전체입력 point 모델)
| 행동 | R² | MAE |
|---|---|---|
| 조향 | 0.998 | 0.0002 |
| 가속 | 0.969 | 0.0317 |
| 브레이크 | **0.937** | 0.0037 |

## 해석
- **브레이크 R²=0.937** — 2024 장거리(제동 거의 없어 퇴화)와 대조적. 합류/분류부는 *실제 제동·상호작용이 풍부*해 브레이크가 학습 가능한 신호가 됨.
- 가속도 0.969로 높음. 단 **전체입력(자차상태 포함)** 결과라 상당부분 *자차상태 연속성(거품)* 기여 — 2024 ablation에서 확인된 패턴.
- **다음 수**: (1) 도로+조건만 ablation으로 거품 분리, (2) **주변차량 위치 특징 추가** — 이 데이터는 매 시점 uv/fv/so 다중차량 행이 있어 *상대위치·간격을 계산해 넣을 수 있음* → 합류 상호작용을 진짜로 학습(거품 완화).
