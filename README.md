# DS-RL-project — 드라이빙 시뮬레이터 가상 피실험자 (BC · CVAE · RL · 하이브리드)

RL Agent of Driving Simulation · **서울시립대학교 교통공학과 첨단교통시설 연구실**

지하도로/고속도로 드라이빙 시뮬레이터 주행 로그로부터 **사람의 운전 행태를 모방하는 정책 모델**을 학습한다.
궁극 목표는 *피실험자 실험을 보완·대체*할 수 있는 운전 에이전트(닫힌 루프 주행 → 새 도로 설계 평가).

> ⚠️ **데이터 비포함**: 이 저장소에는 **코드와 결과(그림·리포트)만** 포함한다.
> 원본 주행 데이터는 인체실험 자료(개인정보·연구 IP)라 공개하지 않는다.
> 스크립트의 데이터 경로는 `<NAS_PATH>` / `<DATA_DIR>` 플레이스홀더로 비워져 있다.

## 1. 개요

- **방법**: Behavioral Cloning (지도학습 기반 모방학습). `상태 → 행동(조향/가속/브레이크)`을 인간 시연으로부터 학습.
- **모델**: 단일프레임 MLP(베이스라인) + 시계열 GRU + 확률적 가우시안 GRU(평균+분산).
- **평가 원칙**: **피실험자 단위 분할** (train/val/test에 서로 다른 사람) → *새로운 운전자에 대한 일반화* 측정.

## 2. 핵심 결과

### (A) 남산터널 데이터 — BC 베이스라인 (held-out 피실험자, GRU)
| 행동 | R² |
|---|---|
| 조향 | **0.999** |
| 가속 | **0.968** |
| 브레이크 | **0.903** |

시간맥락(GRU)이 가속·브레이크 예측의 핵심이었다 (MLP 대비 대폭 향상).

### (B) 2024 지하고속도로 — Ablation: "정확도의 출처"
결정론 모델 R²를 입력 구성별로 비교 (held-out):

| 행동 | 전체 입력 | 도로+조건만 | 자차상태만 |
|---|---|---|---|
| 조향 | 0.998 | **0.842** | 0.996 |
| 가속 | 0.939 | **0.080** | 0.874 |

- **조향**: 도로 기하만으로도 R² 0.84 → *도로를 보고 핸들 꺾는 진짜 신호*가 존재. (새 도로 일반화에 긍정적)
- **가속**: 도로만으론 0.08 → 거의 *현재 속도 유지(자차상태 연속성)*. 장거리 순항 데이터 특성. 차량추종/주변차량 정보가 있어야 의미가 생김.

### (C) 실제 사람 vs BC모델 (2024, held-out)
| 행동 | R² | 지상 | 지하 |
|---|---|---|---|
| 조향 | 0.998 | 0.998 | 0.998 |
| 가속 | 0.939 | 0.920 | 0.946 |

지상/지하 조건 무관하게 충실히 재현. 단 결정론 모델은 **변동성(SDLP의 재료)을 과소평가**하는 경향 → 확률적(가우시안) 모델로 분포까지 학습하면 완화.

### (D) 합류·분류부 통합 (신월여의+복층, held-out, 전체입력)
| 행동 | R² |
|---|---|
| 조향 | 0.998 |
| 가속 | 0.969 |
| 브레이크 | **0.937** |

합류/분류부는 실제 제동·상호작용이 풍부해 **브레이크까지 학습됨**(2024 장거리의 퇴화와 대조). 단 전체입력이라 가속/조향 일부는 자차상태 거품 포함 → 다음: *주변차량 위치 특징 추가*(다중차량 행 존재). 상세: [`reports/report_merge.md`](reports/report_merge.md)

### (E) RL 주행 에이전트 — 닫힌루프 (간이 시뮬, held-out 도로)
파이썬 Gymnasium 간이 운동학 환경(도로는 실데이터에서 재구성, PD 추종검증 RMSE 1.5cm 통과)에서
PPO 에이전트가 **직접 누적 주행**:

| | 이탈율 | 평균속도(m/s) | 사람궤적 RMSE(m) | SDLP(m) | 저크위반 |
|---|---|---|---|---|---|
| 사람 | — | 27.7 | — | **0.221** | — |
| BC(env-native) | 0.00 | 26.9 | 0.463 | 0.332 | 0.000 |
| RL(PPO, 결정론) | 0.00 | 26.9 | **0.318** | 0.084 | 0.000 |
| **RL-σ (보정 확률평가)** | **0.00** | 26.3 | 0.416 | **0.262** | 0.006 |

- RL이 held-out 도로를 **이탈 없이 사람 속도로 완주**. 결정론 평가는 사람보다 '얌전'(SDLP 0.084).
- **RL-σ**: 정책 자체 행동노이즈의 온도 α를 val 도로에서 사람 SDLP에 보정(α*=0.88, **조향 채널만** —
  가속 노이즈는 저크를 깨므로 결정론 유지) → **SDLP 0.262로 사람(0.221) 분포 안에 안착**, 재학습 0.
- 개인추종 RMSE는 e_ref 은닉 설계상 사람 고유 흔들림(≈0.22m)이 바닥 → 인간유사성 주지표는 분포일치.
- **합성 운전자 군집**: 실도로별 (SDLP, LPM, 속도) 특성을 조건별 결합 부트스트랩 → 운전자마다
  (α, 차선편향 b, 속도비 γ) 주입. **사람들 사이의 SDLP 퍼짐까지 재현** (0.221±0.053 vs 0.263±0.057,
  W1=0.043; LPM W1=0.059; 속도는 W1=1.18로 약점). 이탈 0/30 — "합성 피실험자" v1.
- 과정에서 잡은 RL 함정들: 조향 과민(권한 25배)·자살 익스플로잇(이탈=종료 악용)·상태커버리지(발진구간 OOD)·가속채널 노이즈의 저크 파괴 → 상세: [`reports/report_rl.md`](reports/report_rl.md)

### (F) 최종 결과물 — 챔피언 v3.2 하이브리드 + 새 도로 블라인드 검증

**챔피언 v3.2** = 광권한 RL 조향(steer_gain=배포곡률×2.4) + **실제 사람 잔차청크 라이브러리**의
의도-목표 주입 + PD 속도. 판별자 시험(C2ST AUC; 0.5=사람과 구별불가)으로 검증:

| 시험 무대 | AUC | 이탈율 |
|---|---|---|
| 학습 도메인 held-out (2024, v3.2) | **0.671** | 0.00 |
| 미학습 터널(남산), 사람속도 오라클 (v3.2) | 0.769 | 0.03 |
| 미학습 터널, 사람 기록 0% — CVAE 앙상블 속도 (v3.2) | 0.786 | 0.03 |
| **미학습 터널, v3.3(다속도 기반) 제로샷** | **0.656 (사상 최고)** | **0.00** |

**v3.3** = 조향 기반을 다속도·중곡률 학습 정책(rl_multi)으로 교체 — 남산 SDLP
0.423→0.262(사람 0.219), 코너링 바닥 3.4배 개선. 범용 배포 기본값. 조건축(격벽 등)
수정자 모델도 이 기반에서 방향 재현 성립 (§23).

- **"기하만 입력 → 가상 피실험자 군집 → 행태 평가"가 미학습 도로에서 사람 기록 없이
  오라클급으로 작동** — 프로젝트 목표(새 도로 설계안 평가)의 완결 검증.
- 속도 행동 학습(왕숙 다속도 데이터): v_ref 채널 반응성 확보(γ 스윕 단조, 반응비 0.66),
  속도 W1 0.45 m/s (단일속도 기준선 대비 36배 개선).
- **여유율 법칙**(3실험 확증): 조향권한은 배포 도로 최대곡률의 ~2.4배 — 미달 시 정지분지
  붕괴, 과잉(25배) 시 물결 과민.
- **사람다움 상한 캠페인의 정직한 기록**: GAIL 4라운드(전이D/약화D/구간D × MLP/LSTM)와
  증류까지 8연속 실패 → 적대 경로 공식 폐쇄. 성공한 개선은 전부 단순·구조적(실제 사람
  청크, 물리 파라미터)이었다는 것이 이 저장소의 핵심 교훈. 원시궤적 CNN 잣대 기준
  챔피언의 실제 위장력은 0.898(200m)/0.938(1000m) — 상한은 열려 있음.
- **통합 도구**: `34_virtual_cohort.py` — 도로 기하 입력 한 번으로 가상 피실험자 n명의
  주행 CSV + 지표 리포트 생성.

자세한 그림·수치: [`reports/report_rl.md`](reports/report_rl.md) §1~22 (실패 계보 전체 포함).

## 3. 꼭 알아야 할 개념 (한계)

- **열린 루프 vs 닫힌 루프**: 위 R²는 모두 **열린 루프 1-스텝 예측**(매 순간 *실제* 상태를 주고 다음 행동 하나만 예측). 모델이 *직접 누적 주행*하는 **닫힌 루프**에선 작은 오차가 쌓여(compounding error) 벗어날 수 있다 → 높은 정확도 ≠ 자율주행 보장.
- **SDLP/LPM**: 차선위치 *궤적*의 통계라 모델이 직접 달려야(닫힌 루프/간이 차량모델) 모델 SDLP를 구할 수 있다. 열린 루프에선 *실제 사람*의 SDLP만 직접 계산.
- **결정론 모델은 변동성을 줄임** → 사람마다의 차이를 재현하려면 확률적/생성형 정책 필요.

5. ✅ 정량 검증 스위트 — C2ST 판별자 / 동등성 / 조건효과 보존 3층 검정
