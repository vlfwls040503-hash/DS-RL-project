# DS-RL-project — 드라이빙 시뮬레이터 운전행태 모방학습 (Behavioral Cloning)

지하도로/고속도로 드라이빙 시뮬레이터 주행 로그로부터 **사람의 운전 행태를 모방하는 정책 모델**을 학습한다.
궁극 목표는 *피실험자 실험을 보완·대체*할 수 있는 운전 에이전트(닫힌 루프 주행 → 새 도로 설계 평가).

> ⚠️ **데이터 비포함**: 이 저장소에는 **코드와 결과(그림·리포트)만** 포함한다.
> 원본 주행 데이터는 인체실험 자료(개인정보·연구 IP)라 공개하지 않는다.
> 스크립트의 데이터 경로는 `<NAS_PATH>` / `<DATA_DIR>` 플레이스홀더로 비워져 있다.

---

## 1. 개요

- **방법**: Behavioral Cloning (지도학습 기반 모방학습). `상태 → 행동(조향/가속/브레이크)`을 인간 시연으로부터 학습.
- **모델**: 단일프레임 MLP(베이스라인) + 시계열 GRU + 확률적 가우시안 GRU(평균+분산).
- **평가 원칙**: **피실험자 단위 분할** (train/val/test에 서로 다른 사람) → *새로운 운전자에 대한 일반화* 측정.

## 2. 핵심 결과

### (A) 남산터널 데이터 — BC 베이스라인 (held-out 피실험자, GRU)
| 행동 | R² |
|---|---|
| 조향 | **0.999** |
| 가속 | **0.968** |
| 브레이크 | **0.903** |

시간맥락(GRU)이 가속·브레이크 예측의 핵심이었다 (MLP 대비 대폭 향상).

### (B) 2024 지하고속도로 — Ablation: "정확도의 출처"
결정론 모델 R²를 입력 구성별로 비교 (held-out):

| 행동 | 전체 입력 | 도로+조건만 | 자차상태만 |
|---|---|---|---|
| 조향 | 0.998 | **0.842** | 0.996 |
| 가속 | 0.939 | **0.080** | 0.874 |

- **조향**: 도로 기하만으로도 R² 0.84 → *도로를 보고 핸들 꺾는 진짜 신호*가 존재. (새 도로 일반화에 긍정적)
- **가속**: 도로만으론 0.08 → 거의 *현재 속도 유지(자차상태 연속성)*. 장거리 순항 데이터 특성. 차량추종/주변차량 정보가 있어야 의미가 생김.

### (C) 실제 사람 vs BC모델 (2024, held-out)
| 행동 | R² | 지상 | 지하 |
|---|---|---|---|
| 조향 | 0.998 | 0.998 | 0.998 |
| 가속 | 0.939 | 0.920 | 0.946 |

지상/지하 조건 무관하게 충실히 재현. 단 결정론 모델은 **변동성(SDLP의 재료)을 과소평가**하는 경향 → 확률적(가우시안) 모델로 분포까지 학습하면 완화.

자세한 그림·수치: [`reports/`](reports/) 참고.

## 3. 꼭 알아야 할 개념 (한계)

- **열린 루프 vs 닫힌 루프**: 위 R²는 모두 **열린 루프 1-스텝 예측**(매 순간 *실제* 상태를 주고 다음 행동 하나만 예측). 모델이 *직접 누적 주행*하는 **닫힌 루프**에선 작은 오차가 쌓여(compounding error) 벗어날 수 있다 → 높은 정확도 ≠ 자율주행 보장.
- **SDLP/LPM**: 차선위치 *궤적*의 통계라 모델이 직접 달려야(닫힌 루프/간이 차량모델) 모델 SDLP를 구할 수 있다. 열린 루프에선 *실제 사람*의 SDLP만 직접 계산.
- **결정론 모델은 변동성을 줄임** → 사람마다의 차이를 재현하려면 확률적/생성형 정책 필요.

## 4. 로드맵

1. ✅ BC 베이스라인 (남산 / 2024) + ablation + 실제vs모델
2. 🔄 합류·분류부(신월여의/복층) 통합 학습 — 상호작용 풍부, *진행 중*
3. ⬜ **닫힌 루프(간이 차량모델) 구축** → 모델 SDLP/LPM 평가
4. ⬜ 새 도로 일반화 평가 (한 도로/사람을 통째로 held-out)
5. ⬜ (시뮬레이터 프로그램 제어 가능 시) DAgger / 강화학습
